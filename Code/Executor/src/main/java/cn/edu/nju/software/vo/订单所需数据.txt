订单数据：
    需要当天数据（或最近一个交易日数据），默认按时间倒序排列
    格式：时间（hh:mm:ss)、价格、成交量、性质（买入/卖出）
1. 逐笔：
    返回当天每一笔交易的订单数据，默认按时间倒序排列
2. 大单
    返回当天所有大单数据，默认400手以上为大单，可传入参数进行选择，单位为手，默认按交易量倒序排列
3. 分价
    按成交价格将订单分类，返回价格、成交量、占比（该价格下成交量/总成交量），默认按价格倒序排列
4. 分时
    返回时间、成交量、成交价格
    时间为每分钟，成交量包括买入和卖出，价格为这一分钟内的均价，若一分钟内无交易，可返回成交量0
